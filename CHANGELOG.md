# Changelog

## 0.1.0 (2026-05-31)

- Initial release
- Mastodon timeline fetching with pagination
- Bluesky timeline fetching with pagination
- Engagement-based link scoring (post=1.0, boost=0.5, like=0.25)
- Time-decayed scoring
- Per-platform median normalization
- URL title extraction with HTML entity decoding
- Redirect following for short URL dedup
- SQLite caching for posts, URL metadata, and profiles
- Terminal UI with rich formatting and clickable links
- Interactive setup wizard for Mastodon and Bluesky
